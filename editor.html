設定 GitHub 自動儲存

設定後，編輯完點「儲存」會直接 commit 到你的 GitHub repo， Cloudflare Pages 會自動重新部署，不必再手動下載覆蓋檔案。 未設定時，「儲存」則退回下載 JSON 的方式。

建議用 Fine-grained token，只授權此 repo 的 Contents: Read and write 權限。 Token 只存在你的瀏覽器，不會上傳到任何地方。
在其他裝置上「複製設定碼」，到新裝置貼上即可帶入全部欄位（含 Token），不必手動重新輸入。
